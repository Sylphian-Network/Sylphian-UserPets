# Sylphian/UserPets
Give users of your Xenforo forum their own virtual pet.

## Getting started
Check out the [wiki](https://github.com/Sylphian-Network/Sylphian-UserPets/wiki) (Currently in beta) that will explain everything you need to know concerning adding or modifying this add-on.

## Requirements
- Xenforo 2.3.7
- PHP 8.3
- [Sylphian/Library 1.0.6](https://github.com/Sylphian-Network/Sylphian-Library)
- (Optional) [Sylphian/Leaderboard 1.0.0](https://github.com/Sylphian-Network/Sylphian-Leaderboard) Read about using it [here](https://github.com/Sylphian-Network/Sylphian-UserPets/wiki/Leaderboard-Support).
